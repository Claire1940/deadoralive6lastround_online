<Section icon={Map} title="Platform-Specific Instructions">

Each platform handles the cloud and local storage of **dead or alive 6 last round new costumes** differently. Steam users have the easiest path due to the Steam Cloud, while console players may need to juggle USB storage or PlayStation Plus cloud saves.

### PlayStation 5 Requirements
For PS5 users, the transfer can happen on the same console or via cloud storage. If you are moving from a PS4 to a PS5, you must use a USB drive or PS Plus to move the save file first, then let the Last Round application detect it locally.

### Xbox Series X|S Integration
Xbox utilizes the Smart Delivery and Xbox Network ecosystem. As long as you are signed into the same Microsoft account, the game will automatically pull your DLC licenses from the store. Note that on Xbox, you must manually install the DLC packs from the "Manage Game and Add-ons" menu before they appear in-game.

### Steam PC Migration
PC players can use the Steam Cloud or manual file movement. If you are moving to a new PC, ensure "Enable Steam Cloud" is active on both machines. Alternatively, you can copy the folder located at `\Users\(Username)\Documents\KoeiTecmo\DEADORALIVE6` to the new machine.
