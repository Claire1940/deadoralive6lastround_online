<Section icon={Target} title="Step-by-Step: How to Migrate Your Progress">

If you are moving from the 2019 version to the 2026 Last Round update, follow these steps to ensure your data is correctly recognized. Note that you must use the same account (PSN, Xbox Live, or Steam) for the transfer to function.

<Steps>
  <Step num="1" title="Backup Original Data">
    Ensure your original Dead or Alive 6 save data is uploaded to the cloud (PS Plus/Xbox Network) or backed up locally on your PC.
  </Step>
  <Step num="2" title="Install Last Round">
    Download and install Dead or Alive 6 Last Round on your new hardware. Do not create a new save file manually if prompted.
  </Step>
  <Step num="3" title="Initiate Transfer">
    Launch the game. The system should automatically detect the legacy save data. Select "Yes" when asked to import previous progress.
  </Step>
  <Step num="4" title="Verify DLC Licenses">
    Go to the in-game Wardrobe. Check if your previously owned costumes are available. If some are missing, you may need to re-download the "Character Unlock Keys" from the store.
  </Step>
</Steps>

<Callout type="warning" title="Critical Step">
If you have already started a new game in Last Round and wish to import old data, you must delete the existing Last Round save file from your system settings first. The import only triggers if no current-gen save is present.
