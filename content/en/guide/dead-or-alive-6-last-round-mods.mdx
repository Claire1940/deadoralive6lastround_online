<Section icon={Map} title="How to Install Dead or Alive 6 Last Round Mods">

Installing custom assets requires a structured approach to ensure the game engine recognizes the new files without crashing. The most reliable method for managing these modifications is through the REDELBE utility. REDELBE acts as a real-time data injector, loading custom assets directly into the game's memory during runtime. This eliminates the need to permanently modify the game's internal archives, allowing you to toggle modifications on and off with ease.

Before proceeding, make sure your game is updated to the latest version on Steam. Running outdated game versions can cause compatibility issues with modern injector tools.

<Steps>
  <Step num="1" title="Download and Extract REDELBE">
    Download the latest version of the REDELBE utility. Extract the contents of the archive directly into your main game directory where the executable file is located.
  </Step>
  <Step num="2" title="Prepare the Layer2 Directory">
    Locate the folder named **Layer2** inside your main game directory. If this folder does not exist, create it manually. This directory will host all your custom character and texture folders.
  </Step>
  <Step num="3" title="Install Mod Files">
    Download your chosen character or outfit mod. Extract the files into a dedicated subfolder within the **Layer2** directory, maintaining the folder structure specified by the mod creator.
  </Step>
  <Step num="4" title="Configure and Launch">
    Launch the game via Steam. Use the designated in-game hotkeys (usually F2 or F3) during the character selection screen to cycle through and load your newly installed custom models.
  </Step>
</Steps>

<Callout type="warning" title="Steam Compatibility">
If you are using mods that alter gameplay mechanics, character movesets, or frame data, configure your Steam client to run in offline mode to prevent account flags during matchmaking.
</Callout>
